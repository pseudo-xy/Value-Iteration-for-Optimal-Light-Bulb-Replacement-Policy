# Value-Iteration-for-Optimal-Light-Bulb-Replacement-Policy
This project applies value iteration to determine the optimal replacement policy for a fleet of 5,000 light bulbs. Each year, the decision-maker must choose whether to keep aging bulbs running or replace them all, with the goal of minimizing total discounted replacement costs over an infinite planning horizon.

The state variable is the age of the bulbs in years (1 through 6). Replacement costs vary by age, reflecting economies and inefficiencies at different points in the bulb lifecycle (ranging from $16 to $23 per bulb). Bulbs must be replaced by age 6 at the latest — keeping them beyond that is infeasible. A discount factor of 0.95 is applied to future costs, reflecting the time value of money in a long-run infinite-horizon setting.
The value function V(s) represents the minimum total discounted cost from state s onward. Value iteration initializes all state values to zero and repeatedly updates them using Bellman's equation: for each age, the algorithm computes the cost of replacing now (replacement cost plus discounted value of returning to age 0) versus the cost of keeping the bulbs one more year (discounted value of the next age state). The action with the lower cost is selected as the optimal policy at that state. This process repeats until the maximum change across all state values falls below a convergence tolerance of 10⁻⁶.
The output logs value function estimates at selected iterations, showing how V(s) evolves and stabilizes as the algorithm converges. The final results table displays the converged value for each age, the cost of replacing versus keeping, and the optimal action. The summary identifies which ages warrant replacement and which warrant keeping, ultimately prescribing the optimal regeneration cycle — the age at which all bulbs should be replaced to minimize long-run discounted costs.
